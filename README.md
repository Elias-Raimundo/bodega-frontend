Bodega System - Frontend

Frontend desarrollado en Angular para la gestión integral de bodegas, bares y comercios gastronómicos.

Funcionalidades

Dashboard

* Navegación principal
* Gestión multiempresa

Ventas

* Agregar productos al carrito
* Registrar ventas
* Cálculo automático de totales

Mesas

* Apertura de mesas
* Administración de pedidos

Productos

* Alta, modificación y eliminación
* Control de stock

Proveedores

* Alta de proveedores
* Edición y eliminación
* Facturas de compra
* Registro de pagos
* Cuenta corriente
* Estado de deuda pendiente

Reportes

* Consultas generales del negocio

Tecnologías

* Angular 20
* TypeScript
* HTML5
* CSS3
* Angular Router
* HttpClient

Deploy
Frontend desplegado en Render.
